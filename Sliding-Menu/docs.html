cooker.drawer.js

is a jquery plugin for easily implement a sliding drawer into your website.

Drawer GitHub Project Cooker GitHub Project

cooker.drawer.js is included in the Cooker

Author: blivesta

Usage jquery & CSS3

JavaScript


$('.drawer').drawer({
  nav: "drawer-nav",
  navList: "drawer-nav-list",
  overlay: "drawer-overlay",
  toggle: "drawer-toggle",
  openClass: "drawer-open",
  closeClass: "drawer-close",
  desktopEvent: "mouseover", // or click event
  speed: 200,
  width: 280
});

Methods

$().drawer(options)

$().drawer('open')

$('#element').drawer('open')

$().drawer('close')

$('#element').drawer('close')

$().drawer('toggle')

$('#element').drawer('toggle')

$().drawer('resize')

$('#element').drawer('resize')

$().drawer('destroy')

$('#element').drawer('destroy')
